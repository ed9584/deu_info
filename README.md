# DESS-NEXUS
동의대학교 정보를 크롤링해 그를 바탕으로 AI를 사용하여 정보 탐색에 용이한 시스템입니다.

## 주요 기능
1. 동의대학교 웹사이트에서 공지사항 데이터를 크롤링합니다.
2. 크롤링된 데이터를 기반으로 AI를 활용하여 질문에 대한 답변을 생성합니다.
3. Flask를 사용하여 웹 인터페이스를 제공합니다.

## 설치 및 실행 방법
1. 의존성 설치:
    ```bash
    pip install -r requirements.txt
    ```

2. 크롤러 실행:
    ```bash
    python crawler.py
    ```

3. 웹 서버 실행:
    ```bash
    python web.py
    ```

4. 브라우저에서 로컬 서버 접속:
    ```
    http://127.0.0.1:5000
    ```

## 요구 사항
- Python 3.9 이상
- Chrome 브라우저 설치 필요
